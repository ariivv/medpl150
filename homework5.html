Reflection

This image was produced by using adobe illustrator. I used the layer and pen tool to create a fictional image of my eye. The original image I used was taken about 3 years ago, and it was nice to be able to use it in a current project. I didn't have a set plan or color scheme going into the project, but I knew I wanted the colors to contrast each other. I believe these variety of tones work together because of how different they are.
